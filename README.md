## 下载

### Docker

```
docker pull apachecn0/ds100-textbook-zh
docker run -tid -p <port>:80 apachecn0/ds100-textbook-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g dsyilingling-textbook-zh
dsyilingling-textbook-zh <port>
# 访问 http://localhost:{port} 查看文档
```
